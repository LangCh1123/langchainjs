# AWS SageMakerEndpointWithStream

LangChain.js supports integration with AWS SageMaker-hosted endpoints also using streams. Check [Amazon SageMaker JumpStart](https://aws.amazon.com/sagemaker/jumpstart/) for a list of available models, and how to deploy your own.

Here's an example:

```bash npm2yarn
npm install @aws-sdk/client-sagemaker-runtime
```
