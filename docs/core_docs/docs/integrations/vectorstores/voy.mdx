# Voy

[Voy](https://github.com/tantaraio/voy) is a WASM vector similarity search engine written in Rust.
It's supported in non-Node environments like browsers. You can use Voy as a vector store with LangChain.js.

### Install Voy

:::tip
See [this section for general instructions on installing integration packages](/docs/get_started/installation#installing-integration-packages).
:::

```bash npm2yarn
npm install voy-search @langchain/community
```
