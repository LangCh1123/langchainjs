# Faiss

:::tip Compatibility
Only available on Node.js.
:::

Faiss is an in-memory vectorstore that can be saved to a file. It uses [faiss](https://github.com/facebookresearch/faiss).

## Setup

You can install it with

```bash npm2yarn
npm install faiss-node
```
